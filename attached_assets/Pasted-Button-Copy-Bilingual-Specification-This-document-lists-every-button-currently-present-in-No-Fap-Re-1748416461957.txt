Implementation Notes

Centralised copy — Add all keys to locales/zh.json and locales/en.json; hard‑coding strings in components is forbidden.

Hot switching — Changing language via i18n.changeLanguage(lang) must trigger global re‑render; no full reload.

Placeholder guard — If a translation is missing, fall back to a placeholder in the current UI language (e.g. "——"); never mix languages.

Extensible naming — Use unified prefixes (e.g. btn., nav.) for easy bulk search and future languages.

⚠️ Only button‑related copy is in scope for this task; headings, paragraphs, etc. will be internationalised in a separate iteration.
