1. Goal

Replace all placeholder/monochrome icons on the Achievements screen with distinctive, modern‑looking icons that instantly convey each milestone’s spirit. Support both locked and unlocked states.

2. Icon Library & Style

Parameter	Requirement
Library	Phosphor Icons (@phosphor-icons/react) — consistent weight variants + MIT license
Default weight	regular
Unlocked fill & stroke	#5EFCE8→#736EFE radial gradient (Tailwind from-teal-300 via-cyan-400 to-indigo-500)
Locked fill & stroke	Tailwind slate-600/40 (40 % opacity)
Size	w-16 h-16 (64 px) on phones, scales to w-20 h-20 on tablets via responsive classes
Animation	On unlock ➝ 250 ms scale-125 → scale-100 ease‑out with framer-motion
